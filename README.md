📋 InvestiMatch

Software desenvolvido para oferecer recomendações personalizadas de investimentos com base nas preferências financeiras do usuário. Por meio de um questionário simples, que avalia tolerância ao risco, experiência com investimentos, prazo de retorno desejado e necessidade de liquidez, o sistema sugere opções de investimento, como poupança, títulos públicos, ações ou criptomoedas, alinhadas ao perfil financeiro de cada pessoa. O objetivo é auxiliar usuários a tomar decisões de investimento mais informadas e adequadas às suas necessidades.
